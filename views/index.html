Timestamp Microservice

So, you need to know the current time in UTC or Unix formats (or some point in time)?

How to use me:

  1. If you want the current time, make a request to /api/timestamp.
  2. If you want a specific time, make a request to /api/timestamp/:date where :date is a valid date of your choosing.

Example Usage:

  • /api/timestamp/2015-12-25
  • /api/timestamp/1451001600000
  • /api/timestamp/thisisinvalid

Example Output:

If your date is valid: {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

If your date is invalid: {"error":"Invalid Date"}
